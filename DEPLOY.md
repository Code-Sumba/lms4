# Deployment Guide

## Stack
- **Frontend** → Render (Static Site)
- **Backend** → Render (Web Service)
- **Database** → MongoDB Atlas (free tier)

---

## Step 1 — MongoDB Atlas

1. Go to [cloud.mongodb.com](https://cloud.mongodb.com) → create free cluster
2. **Database Access** → Add user (username + password)
3. **Network Access** → Allow from anywhere (`0.0.0.0/0`) for Render
4. **Connect** → Copy the connection string:
   ```
   mongodb+srv://<user>:<password>@cluster0.xxxxx.mongodb.net/motionlms?retryWrites=true&w=majority
   ```

---

## Step 2 — Push to GitHub

```bash
cd d:\lms4
git add .
git commit -m "Ready for Render deployment"
git push
```

---

## Step 3 — Deploy to Render via Blueprint

1. Go to [render.com](https://render.com) → **New → Blueprint**
2. Connect your GitHub repo — Render detects `render.yaml` automatically
3. Two services are created: `lms4-api` (backend) and `lms4-frontend` (static site)

---

## Step 4 — Set Environment Variables

### Backend (`lms4-api`) — Render Dashboard → Environment

| Key | Value |
|-----|-------|
| `MONGO_URI` | your Atlas connection string |
| `CLIENT_URL` | `https://lms4-frontend.onrender.com` *(set after frontend deploys)* |

> `JWT_SECRET` is auto-generated by Render (defined in `render.yaml`).

### Frontend (`lms4-frontend`) — Render Dashboard → Environment

| Key | Value |
|-----|-------|
| `VITE_API_URL` | `https://lms4-api.onrender.com/api` |

After setting env vars, click **Manual Deploy** on each service to rebuild.

---

## Step 5 — Seed the Database (run once)

```bash
cd server
MONGO_URI="mongodb+srv://..." node utils/seed.js
```

---

## Updating After Changes

```bash
git add .
git commit -m "description of change"
git push
# Render auto-redeploys both services on push to main
```

---

## Local Development

```bash
# Terminal 1 — Backend
cd server
cp .env.example .env   # fill in MONGO_URI
npm run dev            # http://localhost:5000

# Terminal 2 — Frontend
cd client
cp .env.example .env.local   # VITE_API_URL=http://localhost:5000/api
npm run dev                  # http://localhost:5173
```

> **Note:** Render free tier spins down after 15 min of inactivity — first request after idle takes ~30 seconds.
